# 🤖 AI Mentor Chatbot

An intelligent chatbot designed to guide students in career decisions, provide emotional support, and assist with goal setting.

## 🚀 Features
- 💬 Interactive chat interface
- 🎯 Career guidance (AI, Data Science, etc.)
- 😊 Mood detection (sad, happy, stressed)
- 🧠 Smart responses based on user input

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript

## 🌍 Vision
This project aims to build an AI-powered personal mentor that helps students grow in career and life by combining technology with human-centered support.

## 🔮 Future Improvements
- 🤖 Real AI integration (API)
- 📅 Daily planner
- 🎯 Career roadmap generator
